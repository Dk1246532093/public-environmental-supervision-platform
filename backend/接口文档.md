## 1. 管理员模块

### 1.1 管理员登录

- **接口地址**: `POST /admins/getAdminsByCodeByPass`
- **请求方式**: POST
- **Content-Type**: `application/x-www-form-urlencoded`
- **接口说明**: 后台管理员登录验证

**请求参数**:

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| adminCode | String | 是 | 管理员登录编码 |
| password | String | 是 | 管理员登录密码 |

**响应参数** (Admins):

| 参数名 | 类型 | 说明 |
|--------|------|------|
| adminId | Integer | 管理员ID |
| adminCode | String | 管理员账号 |
| password | String | 密码 |
| remarks | String | 备注 |

**请求示例**:
```
POST /nepm/admins/getAdminsByCodeByPass
Content-Type: application/x-www-form-urlencoded

adminCode=admin&password=123
```

**响应示例**:
```json
{
  "adminId": 1,
  "adminCode": "admin",
  "password": "123",
  "remarks": null
}
```
